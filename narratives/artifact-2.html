Artifact Description

This artifact is a course planning application originally developed in C++ for CS300. The application reads course data from a file, stores it in a hash table, and allows users to search and display course information, including prerequisites. The enhanced version was recreated in Python with a more modular, object-oriented design.

Justification for Inclusion

I selected this artifact because it demonstrates my understanding of fundamental data structures and algorithms, including hashing, searching, and sorting. It also highlights my ability to evaluate and improve existing implementations while maintaining efficiency and scalability, which are essential skills in software development.

Enhancements Made

The enhanced version transitions the application to Python while preserving the original algorithmic logic. A Python dictionary was used to represent the hash table, improving readability and efficiency. The program was restructured into an object-oriented design using classes such as Course and CourseCatalog, and split into multiple files to improve organization. Additional enhancements included improved error handling and sorting of course data for a more user-friendly display.

Reflection on the Enhancement Process

This process strengthened my understanding of how core data structures can be implemented across different programming languages. One of the key challenges was deciding how closely to replicate the original C++ implementation versus adopting more Pythonic approaches. I chose to leverage Python’s built-in dictionary, which maintained the conceptual integrity of the hash table while improving clarity.

Another challenge was restructuring the application into a modular, object-oriented design. This required careful separation of responsibilities between components, which ultimately resulted in a more scalable and maintainable solution. Overall, this enhancement improved both my algorithmic thinking and my ability to refactor code effectively.

Course Outcomes Addressed

This artifact demonstrates my ability to design and evaluate data structures for efficient storage and retrieval, apply algorithmic problem-solving techniques, and improve existing implementations for better performance and maintainability. It also reflects growth in translating theoretical concepts into practical solutions.
